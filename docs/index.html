About this map

Each pin is a timestamped incident documented somewhere in the UK — pavement footway riding, dangerous road cycling, delivery bikes, and similar behaviour. Clips are anonymised (faces blurred) before publication on YouTube. Original footage is retained privately and may be shared with police when reported via 101.

Only incidents with a published YouTube video appear here. Pins show approximate incident location from device GPS at time of recording — not home addresses.

  • Factual evidence only — no naming or shaming
  • Report danger: 999 · Non-emergency: 101
  • This map does not replace a formal police report
